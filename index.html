Aerial Drones

Missions With 5G And Starlink Support

Context

Unmanned Aerial Vehicle (UAV) are increasingly popular and affordable, making them ideal for various complex tasks such as carrying small loads, scouting areas, and reaching difficult-to-access locations. Leveraging these capabilities, UAV can be invaluable during disaster response scenarios. This project equips a UAV with object detection and avoidance capabilities using a depth sensor camera and a command framework. Additionally, the project enables the UAV to follow a user via coordinates provided by a smartphone application. Furthermore, the UAV serves as a mobile 5G cellular network base station, ensuring constant connectivity. The integration of these technologies enhances UAV functionality and reliability in critical situations.

Goals


Starlink

Connect the drone to the internet using a Starlink access point.

5G Connection

Provide a reliable 5G connection to the end-user.

Camera

Determine the distance of points in front of the drone.

Obstacles avoidance

Create an autonomous way to change the course depending on the surrounding environment.

Live Feed

Give the ability to see what the drone sees in real time.

Follow Mission

Use an app that sends the GPS of the end-user to a mission running on the drone.
