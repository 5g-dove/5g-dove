Aerial Drones

Missions With 5G And Starlink Support

Context

The 5G Dove project is a project that aims to develop a drone capable of flying autonomously, with the help of 5G and Starlink technologies, to perform missions in places where there is no internet connection, such as the Amazon rainforest, for example. This project can also be used in the tourism area, for example, to provide a live feed of a certain places to the tourists.

Goals


Primary

Make a drone network that can share 5G or, in case of a missing connection, share a Starlink connection using 5G


Secondary

Make the mission planner platform more dynamic

Make a Live Feed possible, to use in the tourism area
